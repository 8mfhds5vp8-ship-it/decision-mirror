with governance integrity preserved.

## 2. Prerequisites

Before deployment, ensure:

- [ ] Public repository is created
- [ ] Core documents are present
- [ ] Maintainers are assigned
- [ ] Governance charter is ratified
- [ ] Review workflow is defined
- [ ] Reporting channels are active

Do not proceed if any item is missing.
